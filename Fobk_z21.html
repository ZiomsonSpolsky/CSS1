Co znaczy skrót ID? ID - Identtification, polskie tłumaczenie to identyfikator
Jaki jest cel stosowania ID? Celem stosowania atrybutu ID jest odróżnienie elementów HTML. Można to uzyskać właśnie dzięki identyfikatorowi - ID.
Jakie jest znaczenie znaku # Hashtag najczęściej poprzedzony znakiem # służy do oznaczania tematów na różnych portalach społecznościowych. Sam w sobie znak # pomaga w przegrupowanie danych treści.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie identyfikatora to id="nazwa zdefiniowanego w CSS identyfikatora" czyli: Odpowiedni wpis składa się z selektora, do którego planujemy przypisać identyfikator, oraz z nazwy poprzedzonej znakiem #.
Uwagi na temat stosowania ID(trzy) konieczne wyliczenia.
  • Różnica między id a class -> Identyfikator może wystąpić tylko raz w całym dokumencie, natomiast klasę można powielać do woli.
  • Podobnie jak klasy, identyfikatorów, możemy używać tyle razy, ile razy uznamy to zakonieczne. Działanie takie jest błędem, ale nie wpływa na poprawność działania strony.
  • Elementy oznaczone ID muszą być unikatowe, tzn., że na jednej stronie może wystąpić tylko 1 element z takim ID. Jeśli nie mamy pewności, czy element wystąpi więcej niż raz to możemy bez obaw użyć 'class', które może być przypisane do nieskończenie wielu elementów na stronie.
